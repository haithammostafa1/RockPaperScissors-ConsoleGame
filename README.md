# Stone, Paper, Scissors Game (C++) 🎮

A simple yet well-structured **Command Line Interface (CLI)** game written in C++. This project demonstrates the use of functional programming, custom data types (`structs`), and enumerations (`enums`) to build a clean and readable game logic.

## 📋 Features
- **Interactive Gameplay**: Play against the computer in a multi-round match.
- **Randomized AI**: The computer makes random moves using `rand()`.
- **Result Tracking**: Tracks wins, losses, and draws for each round.
- **Game Summary**: Provides a final result showing the overall winner after all rounds are completed.
- **Replayability**: Option to restart the game immediately after finishing.

## 🛠️ Concepts Covered
- **Enums**: Used for game choices (Stone, Paper, Scissors) and winner types.
- **Structs**: Used to group round information and final game results.
- **Functions**: Modular code design for better maintainability.
- **Control Flow**: Use of `switch` statements and `do-while` loops.
